關於

卡克曼致力於提供客製化服務，包含軟體開發、協同開發、決策智囊、職能諮商，針對不同的狀況，提供適性服務，不只替您開發，更以夥伴的立場為出發點，替您的未來發展設想更多！

秉持著有效解決問題為導向的客製化服務態度，鼓勵客戶在決定是否使用卡克曼的服務之前，先進行諮詢，讓卡克曼了解您的情況後，讓您可以在了解卡克曼能如何提供協助後，在決定是否委託卡克曼幫您解決問題，就不會浪費您的資源，讓需求得到最佳化處理！

著想您是客戶，是朋友，更是夥伴。

卡克曼科技的創立

2020-12

在之前，Btb Lab. 以外部資源的身分，支援一些開發團隊，但對於有挑戰性、有意義，或者具有公益性質的開發深感興趣，後來決定為此創立公司－卡克曼科技有限公司。

Btb Lab. 加入純粹 JavaScript 組件開發

2019-09

在與後端工程師合作開發一段時間後，Btb Lab. 也嘗試使用純 JavaScript開發一些應用，不僅提供前端開發使用，亦能提供同樣使用 JavaScript 開發的後端。

Btb Lab. 加入 Vue 組件開發

2018-11

幸運地，在與使用 Vue 開發的團隊合作後，Btb Lab. 也決定加入 Vue 組件的開發，雖然大多數套件都還沒有發步道NPM，但卻也成為了日後開發的一個邏輯基礎，無論是使用哪一個前端開發框架，或者系統架構所需要的思考能力。

Btb Lab. 加入React組件開發

2018-01

在開始開發React的組件，並發布在NPM作為開源組件前，主要專注在建構開發組件的環境，如Webpack. Gulp, Rollup，在一番努力之後，Btb Lab.的第一個開源組件誕生了－ blacktbox-list.

Btb Lab. 建立

2017-12

於2017年12月時，建立了 Black Tool Box Laboratory 作為一個實驗性質的開發實驗室，並以前端開發作為起始開發點，主要想要開發一些組件、模組、應用的原型，做為日後提供客製化開發的基礎，前端開發只是一個開始，開始實驗一些如何更通用性的開發組件、模組、應用，作為之後客製化開發時，可以減少一些客製化修改幅度或者成本。甚至可以降弟弟三方開發的依賴度，未來也會在適當的時機點，加入後端開發的高客製化支援度的組件、套件或者應用。

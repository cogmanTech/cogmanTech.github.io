卡克曼科技

回顧

BTB 實驗室成立於 2017 年 12 月 1 日，於事隔三年後的同一天，也成立了卡克曼科技，為了記錄卡克曼的成長，每一年的末季，我們都將總結過往一年的成長，並寫成年末回顧。

2022.12.19

全心經營與全力發展

全心經營與全力發展的卡克曼很高興達成了今年的目標，也期待明年會接到什麼特殊的委託。

2021.11.20

成立卡克曼科技有限公司

為了更大的目標，我們成立了公司，也將 BTB Lab. 納入卡克曼科技的一部分，未來我們將做每年的總結與回顧，並以此自我激勵。

2020.10.25

開發 Vue 的組件與重構 React 的組件

今年除了開始 Vue 的組件開發之外，因應新版的 React 推出，原來基於 React 開發的組件，也開始翻新版本。

2019.11.5

開發純 JavaScript 的組件

原本我們利用 React 的特性開發組件，但有些工具類的功能開發，其實並不仰賴框架，因此我們也開發一些跨框架可用的工具組件。

2018.10.28

成立 Black Tool Box Laboratory

為了磨練自己的開發能力，我們成立了開發實驗室，除了從 React 開始，以此為基底開發組件，過程中也嘗試一些不同的開發模式，更重要的是，我們以此訓練如何分析功能與套件的本質。
